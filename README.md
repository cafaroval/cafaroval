# 👋 Hi, I'm Lala Jafarova

I'm a Cognitive Science M.Sc. graduate specialising in **clinical EEG**, 
**source-space modelling**, and **machine learning for neural signal classification**.  
My research focuses on bridging the gap between computational methods and clinical 
neuroimaging — building pipelines that are both scientifically rigorous and practically 
deployable.

Currently based in Münster 🇩🇪 
---

## 🔬 Research Interests

- Epileptiform spike detection · Source localisation · EEG/fMRI connectivity  
- ML classification in biomedical signals · Reproducible neuroimaging pipelines  
- Cognitive control · Chronobiology · Sustained attention
